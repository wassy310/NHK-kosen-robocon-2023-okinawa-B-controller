# NHK-Kosen-Robocon-2023-Okinawa-B-controller

## 概要
高専ロボコン2023、沖縄Bチームで使用したコントローラー(遠隔非常停止、ロボットの状態確認など)  

## 機能一覧
- 遠隔非常停止ボタン
- コントローラとロボットの接続状況確認(Connected or Not-Connected)
- 操作方式確認(手動 or 自動)
- 自動走行時のロボットの進路選択状況の確認(インコース or アウトコース)
- タイマ(試合残り時間の表示)
